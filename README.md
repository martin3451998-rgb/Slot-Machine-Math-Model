# Slot Machine Mathematical Model & Monte Carlo Simulation

This repository contains a self-directed project demonstrating the mathematical modeling and validation of a classic 3-reel slot game. 

# Files in this Repository

**`PAR.xlsx`**: The core mathematical model (PAR sheet). It includes reel strip layouts, symbol distributions, paytable definition, and theoretical calculations for Hit Frequency and Return to Player (RTP) using combinatorics.
**`script.py`**: An Object-Oriented Python script that simulates 1,000,000 spins of the slot machine to validate the theoretical math model empirically using the Law of Large Numbers.

# Game Mechanics
**Reels:** 3
**Paylines:** 1
**Theoretical RTP:** ~95.007%
**Hit Frequency:** ~17.303%

# How to Run the Simulation
1. Ensure you have Python installed.
2. Run the script via terminal
3. The script will output the simulated Total Bet, Total Win, RTP, and Hit Frequency.
